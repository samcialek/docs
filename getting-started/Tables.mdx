## Configure Table Columns

Abisko enables users to interact with tables to configure and sort columns and download data.

### **To add or remove table columns**

- Click on the **Columns** button located above the table on the right.
- Select the columns to hide or display in the table by clicking on the field names.

  <Note>
    Some tables, like the one in the in the Portfolio Dashboard tab (below), also allow users to search for specific values in each column.
  </Note>
  ![Group 1000002817](/images/Group1000002817.png)

### **To download table data**

- Click on the **download** button located next to the **Columns button**

  <Note>
    Some tables allow users to select whether to download data as presented in the table or to download all raw data used to generate the data displayed in the table.
  </Note>

## Interact with Plots

Abisko allows users to interact with different plots and occassionally download data.

### To show & hide data in a plot

- Show and hide data in certain plots by clicking the series within the plot legend.

  <Tip>
    By hiding selected data series, users can focus on more specific trends within a plot. In the image below, fuel data is hidden, allowing the user to quickly analyze monthly electricity trends.
  </Tip>
  ![Group 1000002818](/images/Group1000002818.png)

### To learn more about data points in a plot

- Mouseover different parts of a plot to learn more about the data in the plot

  <Tip>
    In the example below, users can mouseover a point to view details about each property, including the value and associated data completeness. Users can also click on the property name to jump to that property.
  </Tip>
  ![Screenshot 2026 01 06 At 3 42 45 PM](/images/Screenshot2026-01-06at3.42.45PM.png)

### To download data from a plot

- Download data from certain plots by clicking the hamburger menu in the top-right of the plot.

  <Tip>
    In the example below, users can download all the monthly energy consumption values in the plot by clicking on the hamburger menu.
  </Tip>
  ![Group 1000002819](/images/Group1000002819.png)
