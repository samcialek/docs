## Add a new property

<Note>
  All properties shared via **ENERGY STAR Portfolio Manager®** (ESPM) will automatically sync every night with Abisko, ensuring that all property uses and attributes are up-to-date.
</Note>

- **To create a new property**
  1. Click on '[xx] propeties' link in the portfolio banner to open the property list sidebar
  2. Click on 'Add Property' at the top of the sidebar
  4. Add or edit property attributes and save

### Deleting Properties

<Warning>
  To protect against accidental deletions, Abisko does not allow users to directly delete entire properties. Please contact Abisko to request to have a property deleted.
</Warning>

## Edit Property Attributes

To edit the property, use the banner.

### Basic Attributes

Required and optional fields when creating new and editing existing properties.

- **Navigate to Property → Meters → Energy / Water / Waste**
  - **Name** _(required)_: The type of energy or water meter (e.g. electricity, gas, propane).
  - **Country** _(required)_: Custom name for the meter.
  - **Street Address** _(required)_: The units in which the meter entries are input (kWh, therms, liters).
  - **City** _(required)_: If the meter is included in Whole Building performance calculations.
  - **State** _(required for USA & Canada)_: If the meter is included in Whole Building performance
  - **Postal Code** _(required)_: If the meter is included in Whole Building performance calculations.
  - **Property Type** _(required)_: If the meter is included in Whole Building performance calculations.
  - **GRESB Property Type** _(required)_: If the meter is included in Whole Building performance
  - **Year Built** _(required)_: If the meter is included in Whole Building performance calculations.
  - \*\*Gross Floor Area \*\*_(required)_: Self-reported
  - **Date Bought** _(required)_: The date the meter came online and performance metrics begin. Only data after the activation date will be included in performance metrics.
  - **Date Sold** _(optional)_: The date the meter went offline and performance metrics end. Only data before the deactivation date will be included in performance metrics.

  <Note>
    Ownership Period:
  </Note>
  - **Percent Ownership** _(required)_: If the meter is included in Whole Building performance
  - **Gross Asset Value** _(required)_: If the meter is included in Whole Building performance
  - **Data Partners** _(required)_: If the meter is included in Whole Building performance calculations.
