## ESPM sync note

For properties synced from **ENERGY STAR Portfolio Manager® (ESPM)**, meters must be created, edited, or deleted directly in ESPM. Updates will appear in Abisko after the **nightly sync**.

## Create a meter

1. Navigate to **Property → Meters**
2. Select **Energy**, **Water**, or **Waste**
3. Click **Add Meter**
4. Complete the required fields in the sidebar
5. Click **Save**

## Fields (Energy & Water)

- **Meter Type** *(required)*: e.g., electricity, gas, propane
- **Meter Name** *(required)*
- **Units** *(required)*: e.g., kWh, therms
- **Included in Metrics** *(required)*: Yes / No
- **Activation Date** *(required)*: Only data after this date is included in performance metrics
- **Deactivation Date** *(optional)*: Only data before this date is included in performance metrics
- **Notes** *(optional)*

## Additional fields (Waste)

Waste meters include all fields above, plus:

- **Waste Material** *(required)*: e.g., plastics, batteries, electronics
- **Date Entry Method** *(required)*:
  - **Regular**: recurring entries (similar to energy/water)
  - **Intermittent**: infrequent or one-time events (e.g., annual donation of electronics)
  - **Container**: based on container size, empty frequency, and average fill percentage

## Edit or delete a meter

1. Navigate to **Property → Meters**
2. Hover over the meter row
3. Click the action icon that appears on the right
4. Choose:
   - **Edit** to update meter details
   - **Delete** to remove the meter (confirmation required)

> For ESPM-synced properties, edits and deletes must be done in ESPM and will reflect after the nightly sync.
