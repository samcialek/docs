## Bulk Operations

For portfolios with 50+ properties, individual configuration is impractical. Use these bulk tools:

### Bulk Space Type Setup

<Steps>
  <Step title="Download template">
    **Settings** â†’ **Bulk Operations** â†’ **Download Space Types Template**
  </Step>
  <Step title="Fill in spreadsheet">
    One row per property with columns for base building GFA, tenant spaces, and control types.
  </Step>
  <Step title="Upload and validate">
    Abisko checks for errors before applying changes.
  </Step>
  <Step title="Review and confirm">
    Preview changes, then apply to all properties.
  </Step>
</Steps>

### Bulk Meter Assignments

Same workflow applies for linking meters to spaces across your portfolio.

<Tip>
**Template Tip:** Export your current configuration first to see the expected format, then modify and re-upload.
</Tip>

## Custom Emissions Factors

Abisko uses EPA eGRID factors by default. Override for specific scenarios:

| Scenario | How to Override |
|----------|-----------------|
| International properties | **Property** â†’ **Details** â†’ **Custom Factors** |
| Utility-specific data | Enter supplier emissions rate |
| Historical analysis | Apply vintage-appropriate factors |

<Warning>
**GRESB Consistency:** If you use custom factors, document your methodology. GRESB may request verification.
</Warning>
