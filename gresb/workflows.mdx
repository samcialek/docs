Abisko supports several workflows to prepare your portfolio for GRESB submission. Follow these guides based on your specific situation.

## New Submission Workflow

For first-time GRESB reporters:

<Steps>
  <Step title="Connect ESPM">
    Sync all properties from Portfolio Manager
  </Step>
  
  <Step title="Configure spaces">
    Set up landlord/tenant space types for each property
  </Step>
  
  <Step title="Assign meters">
    Link meters to appropriate spaces
  </Step>
  
  <Step title="Review data quality">
    Address gaps, overlaps, and completeness issues
  </Step>
  
  <Step title="Apply gap-filling">
    Estimate missing data where acceptable
  </Step>
  
  <Step title="Export and validate">
    Generate Asset Spreadsheet, review for errors
  </Step>
</Steps>

## Annual Update Workflow

For returning GRESB reporters:

<Steps>
  <Step title="Refresh ESPM sync">
    Pull latest data for new reporting year
  </Step>
  
  <Step title="Review property changes">
    Update ownership periods for acquisitions/dispositions
  </Step>
  
  <Step title="Validate configurations">
    Confirm space types and meter assignments still accurate
  </Step>
  
  <Step title="Check data quality">
    Run data quality dashboard for new year
  </Step>
  
  <Step title="Export and compare">
    Generate new spreadsheet, compare to prior year
  </Step>
</Steps>
