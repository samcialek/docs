## Abisko + ESPM Relationship

```
ESPM (Source of Truth)
├── Property data → Abisko imports
├── Meter readings → Abisko imports
├── ENERGY STAR scores → Abisko imports
│
Abisko (Extended Analysis)
├── Adds landlord/tenant splits
├── Adds space-type assignments
├── Adds scope classifications
├── Exports GRESB-ready data
```

ESPM remains your data source. Abisko adds the analysis layer GRESB requires.
